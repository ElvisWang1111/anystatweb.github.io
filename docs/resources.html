常见错误处理

在使用 Anystat 过程中，您可能会遇到一些常见问题。以下是一些常见错误及其解决方案：

API 相关问题

  • 问题：API key 验证失败 解决方案：请检查 API key 是否正确输入，并确保您的 API key 有足够的余额或使用额度。

  • 问题：模型请求超时 解决方案：请检查网络连接，并尝试减小数据规模或简化请求内容。

数据处理相关问题

  • 问题：数据上传失败 解决方案：请检查文件格式是否支持，文件大小是否超过限制。对于大文件，建议使用路径导入方式。

  • 问题：预处理/建模/可视化代码执行失败 解决方案：Anystat 助手会尝试自动修复 bug，请耐心等待。如果多次尝试后仍失败，可能需要手动调整代码或简化任务需求。

API key 获取途径

Anystat 支持多种大模型 API，您可以通过以下途径获取相应的 API key：

  • Deepseek API key （单击以访问 Deepseek 开放平台）

  • ChatGPT API key （单击以访问 OpenAI Platform，需要信用卡支付）

  • 通义千问 API key （单击以访问阿里云百炼平台，提供一定量免费额度）

  • 智谱AI （单击以访问智谱AI API 调用说明页）

反馈途径

如果您在使用 Anystat 过程中遇到问题或有任何建议，欢迎通过以下途径与我们联系：
